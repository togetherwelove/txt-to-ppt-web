# txt-to-ppt Web Extension

이 프로젝트는 원래 PowerPoint VBA 매크로로 텍스트 파일을 슬라이드로 변환하던 도구를, 브라우저에서 바로 사용할 수 있는 웹 버전으로 확장한 저장소입니다.

## 새로 추가된 웹 버전

웹앱 파일:

- `index.html`
- `styles.css`
- `app.js`

특징:

- `.txt` 파일 업로드 또는 직접 입력 지원
- 텍스트 한 줄마다 슬라이드 1장 생성
- 같은 줄 안의 `/`를 슬라이드 내부 줄바꿈으로 변환
- 빈 줄 유지로 빈 슬라이드 생성 가능
- 브라우저에서 바로 `.pptx` 다운로드
- 배경색, 배경 이미지, 폰트, 폰트 크기 조정 가능
- 텍스트 인코딩은 `UTF-8`로 고정
- 줄바꿈 구분자는 `/`로 고정

## 실행 방법

가장 간단한 방법은 `index.html` 파일을 브라우저에서 여는 것입니다.

1. `txt-to-ppt-main/index.html`을 브라우저에서 엽니다.
2. 텍스트를 붙여 넣거나 `.txt` 파일을 업로드합니다.
3. 필요하면 배경 이미지와 스타일 옵션을 조정합니다.
4. `슬라이드 미리보기`로 분리 결과를 확인합니다.
5. `PPTX 다운로드`를 눌러 프레젠테이션 파일을 생성합니다.

## 텍스트 규칙

예시:

```text
첫 번째 슬라이드 첫 줄/첫 번째 슬라이드 둘째 줄
두 번째 슬라이드 내용

네 번째 슬라이드 내용/마지막 줄
```

동작 방식:

- 입력 텍스트의 한 줄이 슬라이드 하나가 됩니다.
- 같은 줄 안의 `/`는 줄바꿈으로 바뀌어 같은 슬라이드 안에 들어갑니다.
- 빈 줄은 빈 슬라이드로 유지됩니다.

## 배포 방법

정적 파일만으로 동작하므로 아래 같은 서비스에 바로 배포할 수 있습니다.

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

단, 브라우저에서 PPTX 생성을 위해 CDN의 `PptxGenJS`를 사용하므로 인터넷 연결이 필요합니다.
