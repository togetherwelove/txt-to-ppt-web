Browser-Based PPT Generator

텍스트 파일을 바로 프레젠테이션으로 변환하세요

한 줄이 한 슬라이드가 되고, 같은 줄 안의 /는 슬라이드 내부 줄바꿈으로 변환됩니다. 아래 가상 슬라이드에서 텍스트 박스를 드래그해 상하 정렬을 정하고, 패딩과 폰트 스타일을 미리 확인할 수 있습니다.

입력 규칙

  • 텍스트 한 줄마다 슬라이드가 하나씩 생성됩니다.
  • 같은 줄의 /는 줄바꿈으로 들어갑니다.
  • 빈 줄은 빈 슬라이드로 유지됩니다.

입력

파일 업로드 또는 직접 편집

텍스트를 입력하거나 파일을 올리면 미리보기가 바로 반영됩니다.

슬라이드 미리보기

아직 생성된 슬라이드가 없습니다

출력 설정

드래그와 옵션 조합으로 텍스트 위치를 맞출 수 있습니다

가상 슬라이드

텍스트 박스를 드래그하면 상하 정렬이 바뀝니다

정렬: 가운데 / 상단

가사를 입력하면 여기에 표시됩니다
