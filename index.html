텍스트 파일을 바로 프레젠테이션으로 변환하세요

한 줄이 하나의 슬라이드가 되고, 같은 줄 안의 /는 슬라이드 내부 줄바꿈으로 처리됩니다. 아래 가상 슬라이드에서 텍스트 박스를 드래그해 상하 정렬을 바꾸고, 폰트와 패딩 옵션도 함께 조정할 수 있습니다.

입력 규칙

  • 텍스트 한 줄마다 슬라이드가 하나씩 생성됩니다.
  • 같은 줄 안의 /는 줄바꿈으로 들어갑니다.
  • 빈 줄은 빈 슬라이드로 유지됩니다.

입력

텍스트를 입력하거나 파일을 올리면 가상 슬라이드에 바로 반영됩니다.

출력 설정

글자 색상
배경 색상

가상 슬라이드

텍스트를 입력하면 여기에 표시됩니다
